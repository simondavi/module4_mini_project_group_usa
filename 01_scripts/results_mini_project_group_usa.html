Introduction (ALL)

This is an R-Quarto template which can be used for your final assignment task of KODAQS Academy, Course D2 Survey. How to create citable article
It contains prepared sections for each kind of data quality indicators. Choose an appropriate—for data quality related research questions—instrument from the construct Trust or Attitude. For each data quality indicator, you have to select indices that are part of the dataset and/or can be calculated, and specify whether and how you will flag/filter cases. Our final assignment task contains parts that should be worked on in a collaborative way (ALL) and parts that should worked on by individual team members (YOUR NAME). Each teammate should be responsible for at least one indicator section.

Attention Checks (David)

The first kind of data quality indicators are check items. What check items are provided in the dataset and did participants pass the check?

attcheck1.fails <- function(attcheck1){
  n_valid <- sum(!is.na(attcheck1), na.rm = T)
  n_fails <- sum(attcheck1 != "213", na.rm = T)
  return((1 - ((n_valid - n_fails) / n_valid)) * 100)
}


attcheck2.fails <- function(attcheck2){
  n_valid <- sum(!is.na(attcheck2), na.rm = T)
  n_fails <- sum(attcheck2 %in% 2:5, na.rm = T) + sum(attcheck2 == -99, na.rm = T)
  return((1 - ((n_valid - n_fails) / n_valid)) * 100)
}
